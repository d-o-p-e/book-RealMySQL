### 진행 방식

- 각자 미리 정해둔 챕터 읽고 Github에 정리
- 주마다 한 명이 리딩하고, 해당 챕터에 대해서 공유 & 토론
- 경험 공유는 너무 너무 환영입니다


### 주간 리더의 분의 역할

- 스터디 진행!
- 간단한 예제 코드 / 사례 등을 한 번씩 공유해주셨으면 좋겠습니다!!!
- 핵심적인 내용에 대해 관련 아티클 등을 공유


### 정리

- 책의 세부 챕터(x.x.x.x) 단위로 정리
- 파일 네이밍은 주차를 앞에 prefix로 적고, 본인 이름 적기! `[n_본인이름.md]` 예시) `1_천진우.md`
